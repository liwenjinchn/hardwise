Core Chain

01Registry object

位号必须来自 KiCad registry 或 Allegro netlist/BOM identity。

02Deterministic check

规则或 validator 产出 Finding / ValidationReport，不靠模型自由判断。

03Evidence token

结论要落到 `sch:`、`datasheet:`、`bom:` 或 `rule:` token。

04Guarded explanation

模型只能解释工具返回；未知 refdes 返回 structured miss。
